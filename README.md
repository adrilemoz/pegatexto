# Pega Texto
Ferramenta pessoal que extrai o conteúdo principal de uma matéria, removendo ruídos.
Totalmente otimizado para Termux (Sem dependência de navegadores locais).

## Como rodar
### Backend
cd backend && npm install && npm run dev
Roda em http://localhost:4000

### Frontend
cd frontend && npm install && npm run dev
Roda em http://localhost:5173, com proxy /api para o backend.
